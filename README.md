# 🏨 ReadyRoom - Smart Self Check-in

Sistem **Self Check-in tanpa resepsionis** untuk hotel ReadyRoom.

---

## 🚀 Konsep

Customer datang →  
Input kode booking →  
System verifikasi →  
Kamar langsung di-assign →  
Status berubah menjadi ✅ **CHECK-IN**

---

## ✨ Fitur

- Input / scan booking code
- Validasi booking otomatis
- Assign kamar otomatis
- Update status booking (checked_in)
- UI modern & mobile friendly
- Tanpa resepsionis

---

##  Tujuan

Mengurangi:
- Antrian di lobby
- Human error
- Waktu check-in

Meningkatkan:
- Kecepatan pelayanan
- Pengalaman user
- Efisiensi operasional hotel
